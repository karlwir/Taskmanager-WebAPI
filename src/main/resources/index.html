The Taskmanager API

/users

@GET

Get users

Endpoint to get list of users

QueryParams:

  • firstname=<firstname>
  • lastname=<lastname>
  • username=<username>
  • active=true/false
  • sortby=firstname,lastname,username,active
  • sort=asc/desc
  • page=<paginate>
  • pagesize=<pagesize>
    •

@POST

Create user

Endpoint to create a new user

JSON: Required fields are username, firstname and lastname 

                                    
    {
        "username": "anca01",
        "firstname": "Anders",
        "lastname": "Carlsson"
    }

@PUT

Update user

Endpoint to update a user

JSON: Required fields are itemKey, username, firstname, lastname and active 

                                    
    {
        "itemKey": "b2dcb7d9-f66f-41e0-937b-97bb2f0ed165",
        "username": "anca01",
        "firstname": "Anders",
        "lastname": "Carlsson",
        "active": true
    }

@DELETE

Delete user

Endpoint to delete a user

JSON: Required fields are itemKey, username, firstname, lastname and active 

                                    
    {
        "itemKey": "b2dcb7d9-f66f-41e0-937b-97bb2f0ed165",
        "username": "anca01",
        "firstname": "Anders",
        "lastname": "Carlsson",
        "active": true
    }

/users/{itemKey}

@GET

Get user

Endpoint to get one user

/users/{itemKey}/workitems

@GET

Get users work items

Endpoint to get a list with a users work items

@POST

Create new work item and assign to user

Endpoint to create a new work item and assigning it directly to user

JSON: Required fields are title and description 

                                    
    {
        "title": "A short and concise title",
        "description": "A long and verbose description"
    }
